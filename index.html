Realtime CPU Health monitoring

Running a Node.js script on your individual machine we can retrieve realtime performance data via socket.io, the real time CPU performance data is collected and stored within a database. Using our API we return the realtime CPU health to a React.js Front End Component allowing for visualization of data

With Medicus you can keep track of the health of all your machines in one place.
